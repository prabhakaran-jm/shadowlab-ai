# ShadowLab Frontend

Next.js + TypeScript frontend for **ShadowLab – Chaos Engineering for AI APIs**.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Open http://localhost:3000

## Structure

- `app/page.tsx` – main landing / dashboard
- `app/layout.tsx` – root layout
- In the final system: targets UI, scan launcher, reports viewer
